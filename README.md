# Reimagined Armor - Copper & Bronze Compat

A companion resource pack that adds Reimagined Armor-style textures for copper armor (from [Copper Age Backport](https://modrinth.com/mod/copper-age-backport)) and bronze armor (from [Bronze](https://modrinth.com/mod/bronze)).

## Requirements

- **[Armor - Reimagined](https://modrinth.com/resourcepack/armor-reimagined)** by CCF_Ura
- **[Copper Age Backport](https://modrinth.com/mod/copper-age-backport)** (for copper armor)
- **[Bronze](https://modrinth.com/mod/bronze)** (for bronze armor)

## Installation

1. Download the `.zip` from [Releases](https://github.com/YOUR_USERNAME/reimagined-armor-copper-bronze-compat/releases)
2. Place it in your `resourcepacks` folder
3. In the resource pack menu, place this pack **above** `Armor - Reimagined`

## Compatible Mods

| Armor Type | Mod | Textures Replaced |
|---|---|---|
| Copper | Copper Age Backport | `copper_layer_1.png`, `copper_layer_2.png` |
| Bronze | Bronze | `bronze_layer_1.png`, `bronze_layer_2.png` |

## Contributing

Pull requests are welcome. If you'd like to add compat for other armor-adding mods, follow the same structure:

```
assets/minecraft/textures/models/armor/<material>_layer_1.png
assets/minecraft/textures/models/armor/<material>_layer_2.png
```
